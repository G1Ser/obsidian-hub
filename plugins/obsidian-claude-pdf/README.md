# Obsidian Claude PDF

将 Markdown 文件导出为 Claude 风格的精美 PDF 文档。

## 功能特性

- **Claude 风格** - 采用 Claude 官方文档的排版风格，简洁优雅
- **智能分页** - 自动处理标题、表格、图片等元素的跨页问题
- **水印支持** - 可添加自定义水印，保护文档版权
- **格式兼容** - 完美支持 Obsidian 语法，包括 callout、数学公式、代码高亮等
- **实时进度** - 导出过程显示详细进度，体验流畅

## 使用方法

### 导出 PDF

1. 在 Obsidian 中打开要导出的 Markdown 文件
2. 打开命令面板（`Ctrl+P` / `Cmd+P`）
3. 搜索并执行 **Export current Markdown to PDF**
4. 等待导出完成，PDF 文件将保存到指定目录

### 支持的 Markdown 语法

| 语法 | 说明 |
|------|------|
| 标题 | `#` 到 `######` 各级标题 |
| 列表 | 有序列表、无序列表、嵌套列表 |
| 表格 | 标准 Markdown 表格格式 |
| 代码块 | 支持语法高亮 |
| Callout | `> [!info]` 等 Obsidian callout |
| 数学公式 | LaTeX 格式的数学公式 |
| 图片 | 自动适配页面大小 |
| Mermaid | 流程图、时序图等 |
| 内部链接 | `[[链接]]` 格式 |

## 配置指南

打开 **设置 → 第三方插件 → Claude PDF** 进行配置。

### Output

#### Default save folder

PDF 文件的默认保存路径。

- 默认值：桌面
- 说明：可以手动输入路径或点击 Browse 按钮选择文件夹
- 示例：`C:/Users/You/Desktop` 或 `/Users/You/Desktop`

### Watermark

#### Enable

是否在 PDF 中添加水印。

- 默认值：关闭
- 说明：开启后会在每页中心显示水印文字

#### Text

水印文字内容。

- 默认值：`watermark`
- 示例：`机密文档`、`内部资料` 等

#### Color

水印颜色。

- 默认值：`#b0a99e`
- 说明：支持十六进制颜色码
- 示例：`#ff0000`（红色）、`#0000ff`（蓝色）

#### Opacity

水印透明度（0-1）。

- 默认值：`0.2`
- 说明：0 为完全透明，1 为完全不透明
- 推荐：0.1-0.3 之间

#### Font size

水印字体大小（px）。

- 默认值：`80`
- 说明：根据页面大小调整
- 推荐：60-100 之间

### A4 Margins (mm)

PDF 页面的页边距设置。

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| Top | 5 | 上边距 |
| Right | 10 | 右边距 |
| Bottom | 5 | 下边距 |
| Left | 10 | 左边距 |

- 单位：毫米（mm）
- 范围：0-50mm
- 说明：根据实际需求调整，确保内容不被裁剪

## 注意事项

- 仅支持 Obsidian 桌面版
- 需要文件系统访问权限
- 导出过程中请勿关闭 Obsidian
- 大文件导出可能需要较长时间
- 图片会自动缩放以适应页面

## 故障排除

### 导出失败

- 确保已打开 Markdown 文件
- 检查输出目录是否有写入权限
- 尝试重启 Obsidian

### 样式异常

- 检查 Markdown 语法是否正确
- 确认插件版本为最新
- 清除 Obsidian 缓存后重试

### 水印不显示

- 确认水印功能已开启
- 检查透明度设置（可能太低）
- 尝试调整字体大小和颜色
